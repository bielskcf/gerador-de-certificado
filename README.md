Recentemente, enfrentei um desafio: gerar mais de 500 certificados personalizados para o FESTEX 2025 (UnB). Fazer isso manualmente seria inviável e passível de erros.
Desenvolvi um script em Python utilizando as bibliotecas Pandas e Pillow que automatiza todo o processo com apenas o modelo da imagem e a base de dados:
✅ Leitura dinâmica de dados via CSV. 
✅ Tratamento de exceções de codificação (UTF-8/Latin-1/CP1252). 
✅ Quebra de linha inteligente e formatação rica (Bold/Regular). 
✅ Geração de PDFs prontos para impressão em massa.
Resultado: 500+ certificados gerados em apenas 1 minuto. 🚀
A tecnologia serve para potencializar nosso tempo e garantir precisão onde o processo humano é repetitivo.
